# mec01 単体試験仕様書兼結果報告書

## 1. 試験概要

本試験では、Microsoft Entra Connect サーバー（mec01）が設計通りに構築され、
オンプレミス Active Directory と Microsoft Entra ID の同期機能が
正常に動作することを確認する。

本サーバーは Hybrid Identity 構成において
オンプレミスディレクトリとクラウドディレクトリの同期を担う。

---

## 2. 試験対象

|項目|内容|
|---|---|
|サーバー名|mec01|
|役割|Microsoft Entra Connect Server|
|OS|Windows Server 2022|
|ドメイン|corp.local|
|接続DC|addc01 / addc02|
