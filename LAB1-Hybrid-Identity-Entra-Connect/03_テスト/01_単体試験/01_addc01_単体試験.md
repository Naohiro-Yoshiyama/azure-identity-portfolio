# addc01 単体試験仕様書兼結果報告書

## 1. 試験概要

本試験では、Domain Controller（addc01）が設計通りに構築され、  
Active Directory Domain Services が正常に動作することを確認する。

## 2. 試験対象

|項目|内容|
|---|---|
|サーバー名|addc01|
|役割|Domain Controller|
|OS|Windows Server|
|ドメイン|corp.local|
