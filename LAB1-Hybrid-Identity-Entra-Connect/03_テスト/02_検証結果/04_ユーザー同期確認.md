# Entra Connect ユーザー同期試験仕様書兼結果報告書

## 1. 試験概要

本試験では、オンプレミス Active Directory に作成したユーザーが  
Microsoft Entra ID に正常に同期されることを確認する。

本試験により以下を確認する。

・AD → Entra ID 同期  
・属性同期  
・同期サイクル正常動作  

---

# 2. 試験対象

|項目|内容|
|---|---|
|同期サーバー|mec01|
|Stagingサーバー|mec02|
|Active Directory|corp.local|
|Domain Controller|addc01 / addc02|
|同期対象OU|SyncUsers|

---
