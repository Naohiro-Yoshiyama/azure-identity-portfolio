# Entra Connect 結合試験仕様書兼結果報告書

---

# 1 試験概要

本試験では、Active Directory と Microsoft Entra ID の連携により  
ユーザー情報および認証情報が正常に同期されることを確認する。

また、Entra ID および Microsoft 365 へのログインが  
正常に実行可能であることを確認する。

---

# 2 試験対象

|項目|内容|
|---|---|
|Active Directory|corp.local|
|同期サーバー|mec01|
|Stagingサーバー|mec02|
|Domain Controller|addc01 / addc02|
|試験ユーザー|testuser03|

---

# 3 ユーザーログイン試験

|試験ID|試験項目|試験手順|期待結果|結果|エビデンス|
|---|---|---|---|---|---|
|INT-001|ユーザーログイン確認|① addc01 にログインする<br>② Active Directory Users and Computers を開く<br>③ SyncUsers OU を開く<br>④ testuser03 を作成する<br>⑤ mec01 にログインする<br>⑥ PowerShell を起動する<br>⑦ `Start-ADSyncSyncCycle -PolicyType Delta` を実行する<br>⑧ Azure Portal を開く<br>⑨ Entra ID → Users を開く<br>⑩ testuser03 が表示されることを確認する<br>⑪ https://portal.office.com にアクセスする<br>⑫ testuser03 でログインする<br>⑬ ログイン成功画面を確認する<br>⑭ スクリーンショット取得する|ログイン成功||INT-001_login.png|
