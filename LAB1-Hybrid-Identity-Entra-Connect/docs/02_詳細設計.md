## 1. 対象システム構成詳細

### 1.1 構成概要

- 単一フォレスト／単一ドメイン構成  
- サインイン方式：Password Hash Synchronization（PHS）  
- Seamless SSO：有効  
- OUフィルタ：SyncUsers OU のみ同期  
- Active / Staging 冗長構成（Activeは常に1台）  
- カスタム同期ルールなし（既定同期ルールを使用）  

### 1.2 サーバ構成

| サーバ名 | 役割 |
| -------- | ---- |
| aadc01 | Active Directory Domain Services / DNS |
| mec01 | Microsoft Entra Connect (Active) |
| mec02 | Microsoft Entra Connect (Staging) |
| app01 | 検証端末 |

詳細なOS・IP・役割設定は「3.サーバ詳細設計」を参照。
