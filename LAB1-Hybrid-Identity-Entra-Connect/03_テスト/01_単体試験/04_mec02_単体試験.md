# mec02 単体試験仕様書兼結果報告書

## 1. 試験概要

本試験では、Microsoft Entra Connect サーバー（mec02）が設計通りに構築され、
オンプレミス Active Directory と Microsoft Entra ID の同期構成において
Staging モードとして正常に動作することを確認する。

mec02 は Staging サーバーとして構成されており、
同期処理は実行せず、設定同期およびフェイルオーバー待機サーバーとして動作する。

---

## 2. 試験対象

|項目|内容|
|---|---|
|サーバー名|mec02|
|役割|Microsoft Entra Connect Staging Server|
|OS|Windows Server 2022|
|ドメイン|corp.local|
|接続DC|addc01 / addc02|
