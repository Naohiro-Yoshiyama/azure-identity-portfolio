## 1. 対象システム構成詳細

### 1.1 構成概要

- 単一フォレスト/単一ドメイン構成
- サインイン方式：Password Hash Synchronization(PHS)
- OUフィルタ:SyncUsers OUのみ同期
- Active/Staging冗長構成
- カスタム同期ルールなし

## 2. Active Directory 詳細設計

| ドメインFQDN | corp.local |
| NetBIOS名 | CORP |
| フォレスト機能レベル | Windows Server 2016 以上 |
| ドメイン機能レベル | Windows Server 2016 以上 |
| FSMO配置 | addc01 |
| グローバルカタログ | 有効 |
